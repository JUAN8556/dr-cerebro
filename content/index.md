# 🧠 Dr. Cerebro

> *El segundo cerebro de Jano — wiki viva mantenida por IA.*

Inspirado en el sistema de [Andrej Karpathy](https://x.com/karpathy/status/2039805659525644595) y la [implementación de Carlos Azaustre](https://x.com/carlosazaustre/status/2040855361410789749).

Esta wiki no es un archivo de bookmarks. Es una **base de conocimiento viva**: cada enlace, noticia o favorito de redes que entra al sistema se sintetiza automáticamente dentro del artículo temático que corresponde. Yo soy el editor jefe; la IA redacta.

## Cómo navegar

- Usa la **barra lateral** o el **grafo** (abajo a la derecha) para explorar conexiones entre artículos.
- Cada artículo cierra con un bloque de **"Artículos relacionados"** y, más abajo, **backlinks** automáticos.
- El **buscador** (atajo `Cmd/Ctrl + K`) busca por texto completo.

## Cómo funciona

Suelto links en una nota de Apple Notes llamada **"Dr Cerebro Inbox"** desde iPhone o Mac. A las **6:00 AM y 12:00 PM** de cada día, un agente Claude abre cada enlace, lee el contenido real, decide a qué artículo temático pertenece (o crea uno nuevo), y lo sintetiza dentro del texto existente — no como lista de bullets al final. Luego hace `git push` a este repo y GitHub Pages reconstruye el sitio.

Todo el contenido se traduce a español profesional, respetando el tono del autor original. Los enlaces a las fuentes originales quedan siempre disponibles al final de cada artículo.
